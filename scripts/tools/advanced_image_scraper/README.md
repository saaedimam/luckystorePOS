# Lucky Store — Verified Image Scraper

Verifies **name + size + price** match before accepting any scraped image.
Sources (priority order): Shwapno → Chaldal → Google Images.

---

## Setup (run once)

```bash
cd image-scraper
pip install -r requirements.txt
```

---

## Step 1 — Export items from Supabase

Set your credentials:
```bash
export SUPABASE_URL=https://hvmyxyccfnkrbxqbhlnm.supabase.co
export SUPABASE_KEY=your_service_role_key_here
```

Or: copy `all_items.json` from this repo root into this folder:
```bash
cp ../all_items.json .
```

---

## Step 2 — Run the scraper

```bash
python scraper.py
```

This will:
- Search Shwapno, Chaldal, Google for each product
- Score each result: name similarity + size match + price delta
- **Skip LOW confidence matches automatically**
- Download thumbnails to `scraper_output/thumbnails/`
- Save `scraper_output/candidates.json`
- Save `scraper_output/no_match.json` (items with no match found)

**Scrape mode** (edit `SCRAPE_MODE` in scraper.py):
- `"missing_only"` — only items with null image_url (2 items)
- `"all"` — all 536 items (re-verify existing images too) ← **recommended**

---

## Step 3 — Review in dashboard

Open `review_dashboard.html` in your browser:
```bash
open review_dashboard.html      # macOS
start review_dashboard.html     # Windows
```

Dashboard features:
- Side-by-side: **current DB image** vs **scraped match**
- Match score, name similarity %, price delta %
- Filter by confidence (HIGH/MED/LOW), source, status
- **"Approve All HIGH"** button for batch approval
- Multiple candidates per SKU selectable
- Click scraped image → opens source page for verification

---

## Step 4 — Export & Upload

1. In dashboard: click **"⬇ Export Approved"**
2. Copy JSON → save as `scraper_output/approved_images.json`
3. Run uploader:

```bash
python uploader.py
```

This will:
- Upload each approved image to Supabase Storage `item-images` bucket
- Storage path: `items/{SKU}.{ext}`
- Update `items.image_url` in database
- Save `scraper_output/update_log.json`

---

## Match Scoring

| Condition | Points |
|-----------|--------|
| Name similarity ≥ 85% | +4 |
| Name similarity ≥ 72% | +2 |
| Size/weight token exact match | +3 |
| Price within 15% | +2 |
| Price within 30% | +1 |

| Score | Confidence |
|-------|-----------|
| ≥ 7 | HIGH (auto-candidate) |
| 4–6 | MED (manual review) |
| < 4 | LOW (skipped, not shown) |
